# 轻触连点器（Android）

面向 iQOO 12 Pro / Android 设备的本地自动点击工具。无需 Root，不联网，通过 Android 无障碍手势接口执行用户创建的点击与滑动脚本。

## 已实现

- 悬浮控制条：拖动、添加点击点、录制、播放/暂停、停止、参数设置、关闭
- 点击点坐标拖动微调与屏幕编号标记
- 录制点击和单指滑动，并将录制动作同步转发给下层应用
- 循环次数（`0` 为无限循环）、每轮间隔、动作间隔
- 多脚本新建、编辑、运行、删除与本地持久化
- iQOO/OriginOS 的无障碍与后台运行设置引导
- 不读取界面内容，不申请网络权限，所有数据只保存在本机

## 构建

1. 安装 Android Studio（JDK 17、Android SDK 35）。
2. 使用 Android Studio 打开本目录并等待 Gradle 同步完成。
3. 连接手机、启用 USB 调试，运行 `app`；或执行：

   ```powershell
   .\gradlew.bat assembleDebug
   ```

4. APK 位于 `app\build\outputs\apk\debug\app-debug.apk`。

## 手机首次设置

1. 打开应用，点击“打开无障碍设置”。
2. 开启“轻触连点器手势服务”。Android 会显示敏感权限提醒，这是系统对所有此类工具的标准提示。
3. 返回应用，设置脚本并点击“显示悬浮控制条并返回桌面”。
4. iQOO/OriginOS 建议额外允许后台高耗电、关闭对本应用的电池优化，并加入后台白名单，避免锁屏或切换应用后服务被清理。

## 当前边界

- 支持点击和单指直线滑动；暂不录制多指手势、音量键或复杂曲线路径。
- 录制模式使用透明无障碍悬浮层采集手势，再立即转发给底层应用，因此会有很小的转发延迟。
- 银行、支付、密码输入等安全页面可能阻止悬浮层或注入手势，这是系统/应用的安全策略。
- 若计划上架应用商店，需要额外满足各商店对无障碍 API 的声明、审核和隐私披露要求。
